## Running Application
- install java version 8
- either:
    - install maven
        - <code>mvn clean compile</code>
        - <code>mvn exec:java</code>
    - use maven wrapper
        - <code>./mvnw clean compile</code>
        - <code>./mvnw exec:java</code>
- open browser:
  - http://localhost:8081/ctx1/controller
  - http://localhost:8082/ctx2/controller
